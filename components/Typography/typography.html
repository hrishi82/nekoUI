Text Utilities

Documentation and examples of the different text modifications offered by nekoUI.

Headings

HTML headings ranging from h1 to h6 are available.

H1-Heading

H2-Heading

H3-Heading

H4-Heading

H5-Heading
H6-Heading
                        
                            
<!-- Default headings -->
<h1>H1-Heading</h1>
<h2>H2-Heading</h2>
<h3>H3-Heading</h3>
<h4>H4-Heading</h4>
<h5>H5-Heading</h5>
<h6>H6-Heading</h6>
Text sizes

There are various text sizes offered by nekoUI. They can be accessed using the following classes in the order of biggest to smallest font size. text-xl, text-l, text-md, text-reg, text-sm, text-xs

Text Alignment

There are three text alignment options offered by nekoUI. They are left, center and right alignment. They can be accessed using the classes text-left, text-center, text-right

Left Aligned

Center Aligned

Right Aligned

                        
                            
<!-- Text align -->
<h4 class="text-left">Left Aligned</h4>
<h4 class="text-center">Center Aligned</h4>
<h4 class="text-right">Right Aligned</h4>
Text Grey

nekoUI provides to grey out any text using the class text-grey. This can be used where a meaning conveyed by the text has less importance

Greyed text
                        
                            
<!-- Text Grey -->
<h5 class="text-grey">Center text</h5>
