Grid

nekoUI offers four options of grid layout to the user. The options are displayed below.

Grid - 50-50

This grid consists of two columns split in even. Use the class grid-50-50-layout to use it

Cat Toys

Starting At Rs.1299

Explore

    </div>
    <div class="grid-text-content-center">
        <div class="text-container">
            <h2 class="grid-text-heading">Cat Toys</h2>
            <h4 class="grid-text-sub-heading">Starting At Rs.1299</h4>
            <div class="grid-btn-container">
                <a class="btn btn-outline-secondary">Explore</a>
            </div>
        </div>
    </div>
</div>
