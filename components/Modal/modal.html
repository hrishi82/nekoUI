Modal

Modals are positioned over everything else in the document and remove scroll from the page. It can be used to display an important message to the user. Use the class modal to use it

Note: By default, modal comes with a z-index of 100. Modal example:

Title lorem

Lorem ipsum dolor sit amet, consectetur adipisicing elit. Nulla delectus labore aliquid? Dolores officiis mollitia enim dolorum sit

                        
                            
<div class="modal">
    <h4 class="modal-heading">Title lorem</h4>
    <i class="fas fa-times modal-close-btn"></i>
    <p class="modal-description">Lorem ipsum dolor sit amet, consectetur adipisicing elit. Nulla delectus labore aliquid? Dolores officiis mollitia enim dolorum sit</p>

    <div class="modal-btn-container">
        <button class="modal-btn">Action 1</button>
        <button class="modal-btn">Action 2</button>
    </div>
</div>
