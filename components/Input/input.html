Input

Inputs are used in forms to collect user related data

Simple labelled input

Label input has a label present at the top of the input field.

                        
                            
<div class="input">
  <label>Name</label>
  <input class="input-txt" type="text" />
</div>

<div class="input">
  <label>Email</label>
  <input class="input-txt" type="email" />
</div>
Input with validation

These inputs behaves differently based on whether a validation check assigned by the admin is passed or not by the user. There are two classes which you can use depending uppon the state of verfification. For success we use input-success and for error we use input-error

Wrong Password. Try again.

Details Match!

                        
                            
<div class="input">
  <label>Password</label>
  <input class="input-txt input-error" type="password" />
  <p class="input-error-text">Wrong Password. Try again.</p>
</div>

<div class="input">
  <label>Retype Password</label>
  <input class="input-txt input-success" type="password" />
  <p class="input-success-text">Details Match!</p>
</div>
Input Slider

Slider input can be used for range inputs like volume control or for setting the price range

                        
                            
<div class="input">
  <input  type="range"  id="input-id-range"  min="0"
  max="100" value="30"  className="cur-point"/>
</div>
Text Area

Text area input can be used to leave a feedback/note. Its resize feature is disabled by default.

                        
                            
<div class="input">
  <textarea class="textarea" cols="30" rows="10"></textarea>
</div>
Radio buttons

Radio buttons allow the user to select one option from a set.



                        
                            
<div class="radio">
  <input type="radio" name="language" id="html" value="HTML" />
  <label for="html">Option 1</label><br />
  <input type="radio" name="language" id="css" value="CSS" />
  <label for="css">Option 2</label><br />
  <input type="radio" name="language" id="javascript" value="JavaScript" disabled />
  <label for="javascript">Disabled Button</label>
</div>
